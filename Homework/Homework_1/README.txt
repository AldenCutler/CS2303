This program asks the user to inupt a year and then prints out a calendar for that year, accounting for leap years.
To run the file, first compile by typing or copy/pasting 'gcc -Wall calendar.c -o calendar.exe' into the terminal:
Once you've done that, run the executable by typing or copy/pasting './calendar.exe' into the terminal.

I worked on the assigment alone but found an algorithm online for calculating which day of the week any given day is on.
