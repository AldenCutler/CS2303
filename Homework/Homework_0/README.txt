This program takes an integer from the user and returns true if the given integer is a prime number and false if the given integer is not a prime number. 
To run the file prime_number.c: 
1. Compile it by typing or copy/pasting into the terminal gcc -Wall prime_number.c -o prime_number.exe
2. Run the executable file by typing or copy/pasting into the terminal ./prime_number.exe
